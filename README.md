📱 Finance Tracker Android App (Java + XML)
📌 Overview

Finance Tracker is a native Android application developed to help users efficiently manage their personal finances by tracking income, expenses, and budgets. The app provides a clean and intuitive interface to monitor financial activities and gain insights into spending habits.

This project demonstrates strong fundamentals in Android development using Java and XML, along with structured data handling and user-centric UI design.

🚀 Features
💰 Expense & Income Tracking
Record daily transactions with categories like Food, Travel, Bills, etc.
📊 Financial Overview Dashboard
View summarized data of total balance, income, and expenses.

Organize transactions for better financial insights.
